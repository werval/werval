# werval core

```
.
├── io.werval.api               Application Programming Interface, api used by application and test code
├── io.werval.cli               `werval` command line
├── io.werval.commands          Commands shared by the `werval` command line and the build plugins
├── io.werval.devshell          Development mode implementation
├── io.werval.doc               Aggregated javadoc and documentation of werval core archived, used by the website plus
│                               the development mode Plugin that serve /@doc
├── io.werval.runtime           Runtime
├── io.werval.server.bootstrap  Server Bootstrap
├── io.werval.server.jre        com.sun.http Server
├── io.werval.server.netty      Netty Server
├── io.werval.spi               Service Provider Interface, aka. runtime extension points
└── io.werval.test              Test Support
```
